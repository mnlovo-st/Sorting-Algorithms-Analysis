# Sorting Algorithms Performance Analysis 📊

This repository contains the source code and performance analysis for the CSCI 215 Sorting Project. The project evaluates the time complexity and practical execution speed of various sorting algorithms across different input scenarios.

## 🚀 Algorithms Evaluated
* **O(n²) Complexity:** Bubble Sort, Selection Sort, Insertion Sort
* **O(n log n) Complexity:** Merge Sort

## 📁 Repository Structure
* `/source`: Contains the main C++ implementation files.
* `code.txt`: Text version of the source code.
* `*.pptx`: Project presentation slides.
* **Releases**: Check the [Releases section](../../releases) for the final executable and PDF report.

## ⚙️ How to Run
1. Download the latest release from the side panel.
2. Compile the C++ files located in the `source` folder using any standard C++ compiler (e.g., GCC, MSVC).
3. Run the executable to see the performance metrics in the console.
