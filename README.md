# Sorting Algorithms Performance Analysis 📊

This repository contains the source code and performance analysis for the CSCI 215 Sorting Project. The project evaluates the time complexity and practical execution speed of various sorting algorithms across different input scenarios (Random, Sorted, and Descending) using a dataset of 100,000 integers.

## 🚀 Algorithms Evaluated
* **O(n²) Complexity:** Bubble Sort, Selection Sort, Insertion Sort
* **O(n log n) Complexity:** Merge Sort

## 📁 Repository Structure
* `/source`: Contains the main C++ implementation files.
* `code.txt`: Text version of the source code.
* `*.pptx`: Project presentation slides.
* **Releases Section**: Contains the compiled executable and dataset files needed to run the program directly:
  * `SortingProject.exe` (The compiled application)
  * `randomList.txt` (Dataset 1: Random integers)
  * `Sorted.txt` (Dataset 2: Ascending integers)
  * `SortedBackwards.txt` (Dataset 3: Descending integers)

## ⚙️ How to Run
1. Go to the [Releases section](../../releases) and download the compiled `.zip` file containing the executable.
2. Extract the ZIP file into a new folder on your local machine.
3. **Important:** Ensure that `SortingProject.exe` is located in the exact same folder as the three text files (`randomList.txt`, `Sorted.txt`, and `SortedBackwards.txt`).
4. Double-click `SortingProject.exe` to launch the program and view the execution times in the console.
